# 🧠 AI Resume Analyzer & Skill Match System

An AI-powered web application that analyzes resumes and matches them with job descriptions to provide a compatibility score, skill gap analysis, and improvement suggestions.

---

## 🚀 Features

- 📄 Resume Upload (PDF/DOC)
- 🧠 AI-based Resume Parsing
- 🎯 Job Description Matching
- 📊 Match Score Calculation
- ⚡ Skill Gap Identification
- 💡 Suggestions for Resume Improvement
- 🔐 Secure API Handling
